# أسيل بوتيك (Asel Butik) - برنامج إدارة متكامل وموقع المبيعات

برنامج متكامل لبوتيك أزياء نسائية فاخرة، يتكون من واجهتين متصلتين بقاعدة بيانات سحابية متزامنة على **سوبابيس (Supabase)**.

---

## مميزات البرنامج

### 1. واجهة الزبائن الفاخرة (`index.html`)
- **تصميم أبيض وذهبي ملكي:** مظهر أنثوي فاخر وجذاب.
- **تجاوب فائق للهواتف الذكية:** تصفح وتجربة شراء Touch-first سريعة.
- **تفاصيل القطعة والتوفر:** فحص فوري للمخزون حسب المقاس (S, M, L, XL, XXL) واللون المختار.
- **نموذج إتمام الطلب:** يحتوي على قائمة الـ 58 ولاية جزائرية كاملة مع تحديد نوع التوصيل (منزل / مكتب).

### 2. لوحة تحكم المدير (`admin.html`)
- **الطلبات الجديدة (General):** جدول يعرض الطلبات الواردة لحظة بلحظة، مع إمكانية تأكيد الطلب وطباعة الوصل أو إلغائه.
- **أرشيف المبيعات (History):** سجل الفواتير والوصولات المطبوعة للزبائن مع محرك بحث بالاسم أو الهاتف وإمكانية إعادة الطباعة.
- **إدارة المخزون (Stock):** إضافة وحذف وتعديل المنتجات، تحديد الكميات المتوفرة بدقة لكل توليفة من (اللون + المقاس)، ورفع عدة صور لكل منتج.
- **الإحصائيات والتقارير (Analytics):** إجمالي الأرباح، المنتجات الأكثر مبيعاً، رسوم بيانية تفاعلية للمبيعات اليومية وتوزيعها حسب الولايات باستخدام Chart.js.
- **طباعة الوصل:** تصميم فاتورة/وصل مصغر (Bon) مخصص للطباعة الورقية يظهر تلقائياً عند تأكيد الطلب.

## التشغيل والتثبيت المحلي

### المتطلبات الأساسية
- تثبيت [Node.js](https://nodejs.org/) (موصى به للتشغيل التلقائي للخادم).

### خطوات التشغيل
1. **تثبيت الملحقات:**
   ```bash
   npm install
   ```
2. **بدء تشغيل خادم محلي:**
   ```bash
   npm run dev
   ```
   سيقوم الخادم بفتح المتصفح تلقائياً على الرابط: `http://localhost:3000`.

*ملاحظة: يمكنك أيضاً فتح ملفات `index.html` و `admin.html` بالنقر المزدوج عليها مباشرة في أي متصفح دون الحاجة لتشغيل أي خادم، وستعمل قاعدة البيانات ومزامنة الطلبات بشكل كامل!*

## قاعدة البيانات (Supabase)
قاعدة البيانات مستضافة ومفعلة على Supabase. إذا أردت إعادة تهيئة الجداول أو مسح البيانات وإعادتها للافتراضية، يمكنك الضغط على زر **"إعادة تهيئة البيانات"** في أسفل القائمة الجانبية للوحة تحكم المدير، أو تشغيل الأمر التالي في سطر الأوامر:
```bash
node setup_db.js
```
